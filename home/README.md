# home/ - 家サブシステム

プレイヤーの家画面を管理するサブシステム

## 主要ファイル

### **home.py** - 家モジュール
家画面の表示と操作を担当

**役割：**
家では以下の選択肢が提供される：
- 寝る：次の日の朝に時間を進めてmapモジュールへ
- セーブ：ゲームデータを保存
- ロード：ゲームデータを読み込み
- メインメニューへ：main_menuモジュールへ

## 主要クラス

### **HomeModule**
家画面のメインクラス

**主要機能：**
- メインメニューの表示（寝る、セーブ、ロード、メインメニューへ）
- セーブ・ロードスロット選択画面
- 時間管理システムとの連携（睡眠による時間進行）
- セーブ管理システムとの連携

**選択肢：**
```python
choices = [
    {"text": "寝る", "action": "sleep"},
    {"text": "セーブ", "action": "save"},
    {"text": "ロード", "action": "load"},
    {"text": "メインメニューへ", "action": "main_menu"}
]
```

**イベント処理：**
- ↑↓キー：選択肢の移動
- Enter：選択の決定
- ESC：セーブ・ロードモードからの戻る

## 機能詳細

### 睡眠機能
- 時間管理システムの `set_to_morning()` を呼び出し
- 次の日の朝に時間を進める
- mapモジュールへ自動遷移

### セーブ・ロード機能
- 最大10個のセーブスロット（saveslot_01 ～ saveslot_10）
- スロット情報の表示：
  - プレイヤー名
  - 保存日時
  - 空きスロット表示
- ローディング画面の表示

### 描画機能
- メインメニュー画面：タイトル、選択肢、操作説明
- セーブ・ロード画面：5列×2行のスロット表示
- 日本語フォント対応（M PLUS Rounded 1c）

## 連携システム

- **time_manager** - 時間進行の制御
- **save_manager** - セーブ・ロード処理
- **loading_screen** - ローディング画面表示

## UI仕様

**色設定：**
- 背景色：(20, 30, 50) - 濃紺
- テキスト色：(255, 255, 255) - 白
- 選択色：(255, 220, 100) - 黄色
- ボーダー色：(100, 150, 200) - 水色

**フォント：**
- 通常フォント：48px
- 大きいフォント：64px
