## 0. 描画レイヤー順序（DIALOGUE描画スタック）

```
7: UI
6: 演出
─────────────────────────────────────────
5: A   / CGA   装飾（メガネ・ラケット・イヤホン等）┐
4: E   / CGE   顔エフェクト（汗・涙等）            ├ ベース（T or CG）に依存
3: F   / CGF   顔パーツ基本（BRO/CHE/EYE/MOU）    ┘
2: T   / CG    体（どちらか一方が入る）
─────────────────────────────────────────
1: BG           背景
```

> 立ち絵シーン: T + F / E / A  
> CGシーン:     CG + CGF / CGE / CGA  
> システム上はT/CGが同じベーススロットを占有し、上位レイヤーの扱いも共通

## 1. ディレクトリ構成

```
IMAGES/
  01MMK/      ← 桃子（T / F / E / A / CG 系すべてここ）
  02SNK/      ← 沙那子
  07MST/      ← 増田（番号は歯抜けOK）
  BG/         ← 背景
  UI/         ← 演出・UIパーツ
  ICON/       ← アイコン類
  OTHER/      ← キャラ非依存の共有アセット
```

- キャラディレクトリ内はフラット（サブディレクトリなし）
- 拡張子は `.WEBP`（大文字統一）

## 3. 命名規則一覧（レイヤー順）

### 3-1. 背景 (BG) ← レイヤー1

```
BG/BG_[場所]_[時間帯].WEBP
```

| 時間帯 | コード |
|--------|--------|
| 朝     | MNG    |
| 昼     | DAY    |
| 夕方   | EVE    |
| 夜     | NGT    |

```
BG_CLASSROOM_DAY.WEBP
BG_ROOFTOP_EVE.WEBP
```

---

### 3-2. ベースレイヤー ← レイヤー2（T または CG どちらか一方）

#### 3-2a. 立ち絵シーン: 体 (T)

```
[CHAR]_T[ポーズ]_ARM[腕差分]_CLO[服差分].WEBP
```

ポーズは00から順に命名。キャラによってポーズの特徴は異なる。

**服差分 (CLO) 例**

| 番号 | 内容        |
|------|-------------|
| 00   | 制服        |
| 01   | 制服+バッグ  |

```
MMK_T00_ARM00_CLO00.WEBP
MMK_T01_ARM01_CLO01.WEBP
```

#### 3-2b. CGシーン: CG本体

システム上はTと同じベーススロットを占有。  
CG番号が上位レイヤー（CGF/CGE/CGA）との対応キーになる。

```
[CHAR]_CG[番号].WEBP
```

```
MMK_CG00.WEBP
```

---

### 3-3. 顔パーツ・基本 ← レイヤー3（F または CGF）

パーツ別ファイルを重ね合わせ。感情命名なし、パターン番号のみ。

**パーツコード（共通）**

| コード | 部位 |
|--------|------|
| BRO    | 眉   |
| CHE    | 頬   |
| EYE    | 目   |
| MOU    | 口   |

#### 3-3a. 立ち絵シーン: F（Tポーズ番号に対応）

```
[CHAR]_F[ポーズ]_[パーツ][パターン]_[フレーム].WEBP
```

フレームは瞬き・口パク等のアニメ差分。00がデフォルト（00〜02程度）。

```
MMK_F00_BRO00_00.WEBP
MMK_F00_BRO01_00.WEBP
MMK_F00_EYE00_00.WEBP
MMK_F00_EYE00_01.WEBP   ← アニメフレーム違い
MMK_F00_CHE00_00.WEBP
MMK_F00_MOU00_00.WEBP
```

#### 3-3b. CGシーン: CGF（CG番号に対応）

```
[CHAR]_CGF[CG番号]_[パーツ][パターン]_[フレーム].WEBP
```

```
MMK_CGF00_BRO00_00.WEBP
MMK_CGF00_EYE00_00.WEBP
MMK_CGF00_CHE00_00.WEBP
MMK_CGF00_MOU00_00.WEBP
```

---

### 3-4. 顔エフェクト ← レイヤー4（E または CGE）

汗・涙など。内部細分類なし、パターン番号のみ。

#### 3-4a. 立ち絵シーン: E（Tポーズに依存）

```
[CHAR]_E[ポーズ]_[パターン].WEBP
```

```
MMK_E00_00.WEBP   ← T00用エフェクト・パターン0
MMK_E00_01.WEBP   ← T00用エフェクト・パターン1
MMK_E01_00.WEBP   ← T01用エフェクト・パターン0
```

#### 3-4b. CGシーン: CGE（CG番号に依存）

```
[CHAR]_CGE[CG番号]_[パターン].WEBP
```

```
MMK_CGE00_00.WEBP
```

---

### 3-5. 装飾 ← レイヤー5（A または CGA）

メガネ・ラケット・イヤホン等。内部細分類なし、パターン番号のみ。  
常にキャラディレクトリ内に格納。

#### 3-5a. 立ち絵シーン: A（Tポーズに依存）

```
[CHAR]_A[ポーズ]_[パターン].WEBP
```

```
MMK_A00_00.WEBP   ← 桃子・ポーズ00用装飾・パターン0
MMK_A01_00.WEBP   ← 桃子・ポーズ01用装飾・パターン0
```

#### 3-5b. CGシーン: CGA（CG番号に依存）

```
[CHAR]_CGA[CG番号]_[パターン].WEBP
```

```
MMK_CGA00_00.WEBP
```

---

### 3-6. 演出 ← レイヤー6

```
UI/[名前].WEBP
```

---

### 3-7. UI ← レイヤー7

```
UI/UI_[名前].WEBP
```
