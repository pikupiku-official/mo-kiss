# 用語集

本書はドキュメント内の用語を統一するための簡易辞書。

---

## Base State
TITLE / MAIN_MENU / HOME / DIALOGUE / MAP  
ゲームの主画面として動作する状態。

---

## Overlay
Base State の上に開く画面。  
本プロジェクトでは **OPTION** が該当する。

---

## Event
DIALOGUE を起動するストーリー単位。  
`events/E001.ks` のようなスクリプト群を指す。

---

## シナリオフラグ
イベント完了や分岐選択を表す状態。  
原則として DIALOGUE 側で更新される。

---

## 時間/日付
ゲーム内の時刻・日付の進行状態。  
イベント終了時に進行するのが基本。

---

## セーブスロット
セーブデータの単位。  
スロットごとに別世界線として扱う。

---

## プレイヤー名
「はじめから」開始時に決定する名前。  
スロット内で固定。

---

## OPTION
ポーズ型のオーバーレイ。  
BGMは継続し、設定は即時反映される。
