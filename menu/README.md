# menu/ - メインメニューシステム

ゲームのメインメニュー画面を管理するサブシステム

## 主要ファイル

### **main_menu.py** - メインメニュー
メインメニューの表示と操作を担当

**主要機能：**
- ゲーム開始
- タイトルに戻る
- ゲーム終了
- UI操作とイベント処理

### **main_menu_config.py** - メニュー設定
メニューの設定と定数管理

**主要機能：**
- メニューレイアウト設定
- 色設定
- フォント設定

### **ui_components.py** - UI部品
メニュー用のUI部品を提供

**主要機能：**
- ボタンコンポーネント
- テキスト表示
- 選択肢表示
- インタラクティブな要素

## 新機能

- **セーブ・ロード機能：** メインメニューからのセーブ・ロード操作
- **セーブスロット選択：** 複数スロットでの保存・読み込み
- **状態表示：** セーブデータの詳細情報表示
